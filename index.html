Summary

I created a visual prototype for a new kind of wellness app. One that makes the process and documentation so easy, it feels impossible to break the chain of consistency. This app is for those struggling to find their flow in the gym lifestyle.

Problem

The problem that I aimed to save was how workout apps can be overwhelping to new gym goers, so the main mission is developing an easy and aesthetically pleasing process.

Solution

My app tackles this issue head on with a streamlined, fool-proof process designed to get the user gently hooked to the fitness lifestyle.

    Low Fidelity Wireframes and User Flow

    I decided to go with a guided tour to show the user where each button is for the main actions. I kept it at three steps because there are very few buttons in the interface.

    Visual Design

    Throughout my design, I use orange and purple to provide contrast and balance. I also use cirlces in various pages to connect to the main function, tapping the circle to fill the chain. The fonts I used are Trirong for the headings and Rubik for the body copy.
